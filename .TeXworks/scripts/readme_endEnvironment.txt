SCRIPT: endEnvironment.js

VERSION: 0.2

RELEASE DATE:  Wednesday 18th, May, 2011

BRIEF:

\endS{anEnvironment} also indicates any unclosed \begin{Environments}

INSTALL:

(The zip completely replaces any version released here previously. 
If you installed one, delete its whole directory first.)

Unzip to any directory under you TeXworks script directory - Use Menu Scripting/Scripting TeXworks/ Show Script Folder to open the Scirpts' main folder

Change the Alt key combination if you want/need to avoid conflicts with any existing scripts.

Then use Tw Menu item, Scripts/Scripting TeXworks/Reload Scripts List

USAGE:

Position cursor where you want to end an environment, use Alt E twice

UPDATES:

http://TwScript.PaulANorman.com/downloads

BACKGROUND:

Sometimes people have asked for a shortcut key for ending LaTeX Environments. This also indicates which environments are still unclosed, showing a dropdown box to choose from, especially useful in complex structures.
